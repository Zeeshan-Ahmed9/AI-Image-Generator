# AI Image Generator

A Next.js-based AI image generation application with a modern UI and backend API using Node.js, Express, and MongoDB.

## Features

- **AI Image Generation**: Generates images based on text prompts using Clipdrop API.
- **User Authentication**: Secure signup, login, and session management.
- **Credit System**: Users consume credits to generate images.
- **Transaction History**: Track all image generation transactions.
- **Responsive Design**: Modern, glassmorphism-based UI with dark mode.
- **Social Authentication**: Login with Google.

## Tech Stack

- **Frontend**: Next.js 14, React 18, Tailwind CSS
- **Backend**: Node.js, Express, Typescript
- **Database**: MongoDB
- **AI**: Replicate AI
- **Authentication**: JWT, Google OAuth

## Prerequisites

- Node.js (v16 or higher)
- MongoDB
- A Clipdrp AI API Key
